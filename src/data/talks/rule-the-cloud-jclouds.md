Apache jclouds is an open source multi-cloud toolkit for the Java platform that gives you the freedom
to create applications that are portable across clouds while giving you full control to use cloud-specific features.

In this talk I explored the core concepts around jclouds and did a demo showing how the same code can be used to
manage your infrastructure in different cloud providers.
